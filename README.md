# 🌑 Dark Hours of India
## Accidental Deaths Pattern Analysis in India

A multi-factor analysis of accidental deaths across Indian states — 
correlating weather, time, location, and socioeconomic factors using 
Python, SQL, and Power BI.

## 🎯 Objective
Analyze accidental death patterns in India to uncover hidden correlations 
with weather, season, time, geography, and socioeconomic conditions.

## 📊 Data Sources
- NCRB — Accidental Deaths & Suicides in India (ADSI)
- IMD / OpenWeatherMap — Weather data
- Census India — Socioeconomic indicators

## 🛠️ Tech Stack
- Python (Pandas, Matplotlib, Seaborn, Plotly, Folium)
- SQL (SQLite)
- Power BI
- Jupyter Notebook
